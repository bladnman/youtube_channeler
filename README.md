# YouTube Channeler

## About
This is a simple app that allows you to search for a YouTube channel and then see the videos from that channel.

![Day 1 Screenshot](./docs/images/day1_screenshot.png)


## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
